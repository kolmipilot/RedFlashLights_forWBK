# Tactical Red Lights for WBK

WBK Red Light Pack is a lightweight addon that expands the functionality of WebKnight's flashlight system by adding realistic low-visibility red lights for tactical operations.

The mod introduces two red-light variants:

* Red Headlamp
* Red Shoulder Light

These lights are designed for close-range illumination while minimizing visibility to other personnel. Red light helps preserve night vision and reduces the risk of detection compared to standard white flashlights, making it ideal for navigation, map reading, vehicle operations, and other low-light tactical tasks.

## Features

* Realistic low-intensity red illumination
* Head-mounted red light
* Shoulder-mounted red light
* Preserves night vision
* Reduced visual signature compared to white lights
* Multiplayer compatible
